# Technitium DNS Server

[Technitium DNS Server](https://technitium.com/dns/) is Self host a DNS server for privacy & security.

## Introduction

This chart bootstraps an [Technitium DNS Server](https://hub.docker.com/r/technitium/dns-server/) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.
